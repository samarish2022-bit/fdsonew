# Сайт Федерации дартс Самара

Веб‑приложение для отображения рейтингов игроков, календаря соревнований, новостей и медиа‑галереи по дартсу в Самарской области.  
Проект состоит из статических HTML‑страниц, стилей и скриптов, а также простого Node.js‑сервера на Express для раздачи контента и обработки служебных задач.

## Структура проекта

- `index.html` — главная страница сайта
- `rating.html`, `rating-*-detail.html` — рейтинги и подробные страницы игроков
- `competitions.html` — календарь и список соревнований
- `news.html` — раздел новостей
- `gallery.html` — фотогалерея
- `admin.html` — административный интерфейс (управление данными)
- `css/` — стили
- `js/` — клиентские скрипты
- `images/` — изображения
- `data/` — JSON‑данные для сайта
- `server.js` — Node.js‑сервер на Express
- `scripts/`, `vendors/` — служебные скрипты и сторонние библиотеки

## Запуск проекта локально

Требуется установленный Node.js (LTS).

```bash
npm install
npm start
```

По умолчанию сервер поднимается на `http://localhost:3000` (или другом порту, указанном в `server.js`).

## Зависимости

Основные зависимости описаны в `package.json`:

- `express` — HTTP‑сервер
- `sharp` — обработка изображений (оптимизация картинок)

Установка:

```bash
npm install
```

## Разработка

- Верстка — адаптивная, ориентирована на современные браузеры.
- Статические данные хранятся в папке `data/` в формате JSON.
- Клиентские скрипты в `js/` отвечают за загрузку и отображение данных (рейтинги, соревнования и т.д.).
- В `scripts/` могут находиться дополнительные утилиты, например `optimize-hero` для оптимизации изображений.
